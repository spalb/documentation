How to install and use 
===============================

1.Install this package by run the following command.

.. code::

   pip install spalb

2.Import this package by run the following command.

.. code::

   import spalb

3.Choose the desired function and run with command in the following format:

 * spalb.function_name(‚Äúfilename‚Äù)

Example of command is shown below.

.. code::

   spalb.rpw("data.txt)

4.When you run command in step 3 it show message ‚ÄúEnter the cycle time‚Äù. 

.. code::

   Enter the cycle time :

You must enter the cycle time for the function to be able to solve the problem.

 * lcr, rpw, comsoal, u_ga ü†í Cycle time must larger than maximum processing time
 *	parallel_ga ü†í Maximum processing time cann't exceed twice the cycle time

 * lcr, rpw, comsoal, u_ga ü†í Cycle time is less than maximum processing time
 *	parallel_ga ü†í Maximum processing time exceed twice the cycle time

You must enter the new cycle time correctly for the function to be able to solve the problem.
